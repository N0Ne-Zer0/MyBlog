## A
{% note info %}
### 问题陈述

您将获得正整数 $N$ 和 $M$ 。

如果重复执行以下操作，而 $M$ 的值不是 $0$ ，请找出执行的操作次数。

- 设 $x$ 为 $N$ 除以 $M$ 的余数。将 $M$ 的值替换为 $x$ 。

可以证明，经过有限次数的运算后， $M$ 变成了 $0$ 。
{% endnote %}

{% note primary %}
#### 思路:
按照题意模拟并计数即可。
{% endnote %}

**Code**
```cpp
void sol()
{
    int M,N;
    cin>>N>>M;
    int cnt=0;
    while(M)
    {
        M=N%M;
        cnt++;
    }
    cout<<cnt;
}
```

## B
{% note info %}
### 问题陈述

$xy$ \平面上有两个圆 $C_1$ 和 $C_2$ 。在这个问题中，圆指的是圆周而非圆面。  
圆 $C_1$ 的圆心为 $(X_1, Y_1)$ ，半径为 $R_1$ 。  
圆 $C_2$ 的圆心为 $(X_2, Y_2)$ ，半径为 $R_2$ 。  
确定圆 $C_1$ 和 $C_2$ 是否有公共点。换句话说，判断是否存在至少一个到 $(X_1, Y_1)$ 的距离为 $R_1$ 且到 $(X_2, Y_2)$ 的距离为 $R_2$ 的点。

您获得了 $T$ 个测试用例；逐一解决。
{% endnote %}

{% note primary %}
#### 思路:
题意为判断两圆是否相交，即**不相离**、**不内含**。
{% endnote %}

**Code**
```cpp
void sol()
{
    int X1,Y1,R1,X2,Y2,R2;
    cin>>X1>>Y1>>R1>>X2>>Y2>>R2;
    if((X1-X2)*(X1-X2)+(Y1-Y2)*(Y1-Y2)<=(R1+R2)*(R1+R2)&&(X1-X2)*(X1-X2)+(Y1-Y2)*(Y1-Y2)>=(R1-R2)*(R1-R2))
    {
        cout<<"Yes\n";
    }
    else cout<<"No\n";
}
```

## C
{% note info %}
### 问题陈述

寿司的原料有 $N$ 块醋饭和 $M$ 块内塔（配料）。

第 $i$ 个shari的权重是 $A_i$ ，第 $j$ 个neta的权重是 $B_j$ 。

您将结合 Shari 和 Neta 制作寿司。

要制作一件寿司，您需要将一份 Shari 和一份 Neta 结合在一起。这里， Neta 的重量最多是 Shari 重量的两倍。此外，相同的 Shari 或 Neta 不能用于多份寿司。

求最多可以制作多少个寿司。
{% endnote %}

{% note primary %}
#### 思路:
对于第$i$个Shari和第$j$个Neta，满足$A_i\times 2\ge B_j$时，才能结合在一起。
因此我们将$A$和$B$排序后，利用指针$i$和$j$，分别对$A$和$B$进行扫描：
* 如果当前的$i$和$j$不满足条件，那么就将$i$向后移动。
* 如果当前的$i$和$j$满足条件，那么就将他们组合成寿司，$ans$累加，并将$i$和$j$均向后移动，防止重复使用。（注意越界判定）
{% endnote %}

**Code**
```cpp
void sol()
{
    int N,M;
    cin>>N>>M;
    for(int i=1;i<=N;i++)cin>>A[i];
    for(int i=1;i<=M;i++)cin>>B[i];
    sort(A+1,A+N+1);
    sort(B+1,B+M+1);
    int j=1,ans=0;
    for(int i=1;i<=N;i++)
    {
        if(B[j]>2*A[i]&&j<=M)continue;
        if(j>M)break;
        ans++;
        j++;
    }
    cout<<ans;
}
```

## D
{% note info %}
### 问题陈述

有一个包含 $H$ 行和 $W$ 列的网格。从上数第 $i$ 行、从左数第 $j$ 列的单元格称为单元格 $(i, j)$ 。

每个细胞都是白色或黑色的。网格由 $H$ 字符串 $S_1, S_2, \ldots, S_H$ 描述，每个字符串的长度为 $W$ 。如果 $S_i$ 的第 $j$ 个字符是 `.`，则单元格 $(i, j)$ 为白色；如果 $S_i$ 的第 $j$ 个字符是 `#`，则单元格 $(i, j)$ 为黑色。

您执行以下操作 $10^{100}$ 次。

- 同时将以下规则应用于所有单元格。
    - 当且仅当存在至少一个与其相邻的黑色单元格时，操作前为白色的单元格才会变为黑色。这里，单元格 $(x, y)$ 和 $(x', y')$ 是相邻的，当且仅当其中一个单元格位于另一个单元格的 $8$ \-邻域内，即 $\max(|x-x'|, |y-y'|) = 1$ 。
    - 操作前为黑色的单元格变为白色。

求操作后每个单元格的颜色。
{% endnote %}

{% note primary %}
#### 分析:
每次操作会使**黑格子**变白格子，并且将周围**有黑格子的白格子**变黑。执行$10^{100}$次操作，显然我们无法模拟这么多次，我们只能找变化规律。
* 显然，当白格子周围有黑格子时，经过偶数次变化，两者颜色不变；经过奇数次变化，两者颜色交换。（以下简称其为**互锁格子**）

我们找到所有互锁格子后，剩下的非互锁格子**必定全为黑色或白色**，且这些互锁格子的边缘一定是相应的黑色或白色。
* 若全为黑色，则经过一次变换后，就会转变成了全为白色的情况。（后续以全白说明）

由于互锁格子会根据操作次数不断变化，当外侧变为黑色时，下一次变化就会将黑色**向外传递**到非互锁格子，而再下一次，会再往外传递（因为外面为白色），如此就会像波一样不断向外扩散，而互锁格子就如同脉冲信号一样，不断间歇地产生波，周期为2，那么对于波经过偶数次的格子，其颜色仍为白色，经过奇数次的格子，其颜色变黑。
而在网格上，实际上就是判断非互锁格子距**离其最近**的黑色格子的**切比雪夫距离**的奇偶性。
*切比雪夫距离：$d_{\text{Chebyshev}}((x_1, y_1), (x_2, y_2)) = \max(|x_1 - x_2|, |y_1 - y_2|)$*

#### 思路：
考虑到非互锁格子可能为黑色，而黑色又比较特殊，所以我们需要先模拟进行一次操作。
我们找到所有周围有黑色格子的白色格子并记录下来后，将所有黑色格子变为白色，再将记录下来的白色格子变黑，就实现了一次操作。
求**切比雪夫距离**就用多源BFS计算即可。
{% endnote %}

**Code**
```cpp
void sol()
{
    int H,W;
    cin>>H>>W;
    for(int i=1;i<=H;i++)
    {
        cin>>S[i];
        S[i]=" "+S[i];
    }
    vector<vector<int>>dis(H+5,vector<int>(W+5,-1));
    queue<pair<int,int> >q;
    for(int i=1;i<=H;i++)
    {
        for(int j=1;j<=W;j++)
        {
            if(S[i][j]=='#')continue;
            bool fg=0;
            for(int k=0;k<8;k++)
            {
                int sx=i+x[k],sy=j+y[k];
                if(sx<1||sy<1||sx>H||sy>W)continue;
                if(S[sx][sy]=='#')
                {
                    fg=1;
                    break;
                }
            }
            if(fg)q.push({i,j});
        }
    }
    for(int i=1;i<=H;i++)
    {
        for(int j=1;j<=W;j++)
        {
            if(S[i][j]=='#')S[i][j]='.';
        }
    }
    while(q.size())
    {
        auto [tx,ty]=q.front();
        q.pop();
        S[tx][ty]='#';
    }
    for(int i=1;i<=H;i++)
    {
        for(int j=1;j<=W;j++)
        {
            if(S[i][j]=='#')
            {
                dis[i][j]=0;
                q.push({i,j});
            }
        }
    }
    if(q.empty())
    {
        for(int i=1;i<=H;i++)
        {
            for(int j=1;j<=W;j++)
            {
                cout<<'.';
            }
            cout<<'\n';
        }
        return;
    }
    while(q.size())
    {
        auto [tx,ty]=q.front();
        q.pop();
        for(int i=0;i<8;i++)
        {
            int sx=tx+x[i],sy=ty+y[i];
            if(sx<1||sy<1||sx>H||sy>W||dis[sx][sy]!=-1)continue;
            dis[sx][sy]=dis[tx][ty]+1;
            q.push({sx,sy});
        }
    }
    for(int i=1;i<=H;i++)
    {
        for(int j=1;j<=W;j++)
        {
            cout<<(dis[i][j]%2==0?'.':'#');
        }
        cout<<'\n';
    }
}
```

## E
{% note info %}
### 问题陈述

对于正整数 $a$ 和 $b$ ，将 $\mathrm{concat}(a, b)$ 定义为依次写入 $a$ 和 $b$ 形成的整数。更正式地说， $\mathrm{concat}(a, b)$ 定义如下。

- 令 $A$ 和 $B$ 分别为通过十进制写入 $a$ 和 $b$ 形成的字符串。令 $C$ 为按顺序连接 $A$ 和 $B$ 形成的字符串。将 $C$ 解释为十进制整数的值为 $\mathrm{concat}(a, b)$ 。

例如，如果 $a = 123$ 和 $b = 45$ ，则为 $\mathrm{concat}(a, b) = 12345$ 。

您将获得正整数 $N$ 和 $M$ 。  
求不大于 $N$ 的正整数对 $(x, y)$ 的模 $998244353$ 的数量，使得 $\mathrm{concat}(x, y) \equiv x + y \pmod{M}$ 。

您获得了 $T$ 个测试用例；逐一解决。
{% endnote %}

{% note primary %}
#### 思路:
数论题，推导过程如图：
<img src="image.png" alt="推导过程" width="50%">
$L(x)$表示x的长度，如$L(325799)=6$。

*在本题中，由于数据很大，需要用到**unsigned long long**，并且$\frac{N}{\frac{M}{G}}$不能写成$\frac{N\times G}{M}$，否则数据会溢出*
{% endnote %}

**Code**
```cpp
int _10[20];

int L(int x)
{
    int res=0;
    while(x){x/=10;res++;}
    return res;
}
void sol()
{
    int ans=0;
    int N,M;
    cin>>N>>M;
    for(int i=1;i<=L(N);i++)
    {
        int G=__gcd(M,_10[i]-1);
        int sumx=N/(M/G);
        int sumy=9*_10[i-1];
        if(i==L(N))sumy=N-_10[i-1]+1;
        ans=(ans+sumx%mod*(sumy%mod)%mod)%mod;
    }
    cout<<ans<<'\n';
}

signed main()
{
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int T=1;
    _10[0]=1;
    for(int i=1;i<=19;i++)_10[i]=_10[i-1]*10;
    cin>>T;
    while(T--)sol();
}
```

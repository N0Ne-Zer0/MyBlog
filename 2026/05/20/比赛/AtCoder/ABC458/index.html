给你一个由小写英文字母和正整数 NNN 组成的字符串 SSS 。 SSS 的长度至少为 2N+12N+12N+1 。
求从 SSS 的开头删除 NNN 个字符和从结尾删除 NNN 个字符后得到的字符串。

思路：

设SSS长度为LLL，输出SSS的第N+1N+1N+1到第L−NL-NL−N个字符即可，由于SSS下标从000开始，即输出SNS_NSN​至SL−N−1S_{L-N-1}SL−N−1​。

问题陈述

有一个网格，网格中有 HHH 行和 WWW 列。从顶部起第 iii 行，从左侧起第 jjj 列的单元格称为单元格 (i,j)(i, j)(i,j) 。
当 ∣x1−x2∣+∣y1−y2∣=1|x_1 - x_2| + |y_1 - y_2| = 1∣x1​−x2​∣+∣y1​−y2​∣=1 时， (x1,y1)(x_1, y_1)(x1​,y1​) 和 (x2,y2)(x_2, y_2)(x2​,y2​) 单元格被称为相邻边。
请找出与每个单元格边相邻的单元格数。

思路：

对于一个格子(i,j)(i,j)(i,j)，设它的相邻格数为4，如果它靠近上下边缘则减1，靠近左右边缘则减少1，显然最小数为2，遍历即可。

问题陈述

给你一个由大写英文字母组成的字符串 SSS 。
请找出满足以下所有条件的 SSS 的子串（连续子序列）的个数。

  • 由奇数个字符组成。
  • 它的中间字符是 C。更正式地说，如果提取的子串由 lll 个字符组成，那么它的 ((l+1)/2)((l+1)/2)((l+1)/2) 个字符就是 C。
    即使两个子串作为字符串是完全相同的，如果它们是从不同的位置提取出来的，也会被分开计算。

思路：

由于只求字符CCC为中心的连续子序列个数，我们只考虑每个CCC对答案的贡献。
设SSS长度为LLL，对于某一个CCC，如果他的下标为i，即Si=CS_i=CSi​=C时，对答案的贡献为从iii到最左侧000，以及到最右侧L−1L-1L−1的距离取最小值，也就是ans+=min⁡(i+1,L−i)ans+=\min{(i+1,L-i)}ans+=min(i+1,L−i)。

D

问题陈述

黑板上写着一个整数 XXX 。

给你 QQQ 个查询，让你按顺序处理。 iii -th 查询 (1≤i≤Q)(1 \le i \le Q)(1≤i≤Q) 如下。

给出两个整数 AiA_iAi​ 和 BiB_iBi​ 。在黑板上写下两个新的整数 AiA_iAi​ 和 BiB_iBi​ 。
然后，输出写在黑板上的 2i+12i+12i+1 个整数的中位数。

分析：

黑板上初始有一个整数XXX，随后每次增加2个数字，并询问黑板上数字的中位数是多少。如果每次询问都插入后排序求结果的话，时间复杂度为O(Q2log⁡Q)O(Q^2\log Q)O(Q2logQ)，显然无法通过该题，那么我们换个思路。
由于每次只加入两个数字A,BA,BA,B，假设在加入前数列有序，中位数为MMM，那么有三种情况：

  • 当AAA和BBB均小于MMM，则中位数会左移一位。
  • 当AAA和BBB均大于MMM，则中位数会右移一位。
  • 当AAA和BBB其中一个大于MMM，另一个小于MMM，则中位数不变，仍为MMM。

实际上，中位数只会每次移动一个位置，添加数后需要找到和MMM相邻的数即为答案（保证有序）。

思路：

由于中位数两边均有序，我们可以维护两个大小相同的堆，小于中位数则放入大根堆，大于中位数则放入小根堆，随后平衡两者的大小，堆顶即为答案。（由于总数为奇数个，我们可以复制一份堆顶放入大小根堆，在平衡前弹出并保留）
tips：对于小根堆的处理，可以直接在入堆时取反，取出时再取反即可。

复杂度O(Qlog⁡Q)O(Q\log Q)O(QlogQ)

问题陈述

求长度为 X1+X2+X3X_1+X_2+X_3X1​+X2​+X3​ 的序列 A=(a1,⋯ ,aX1+X2+X3)A = (a_1, \cdots, a_{X_1 + X_2 + X_3})A=(a1​,⋯,aX1​+X2​+X3​​) 中，满足以下所有条件的模数 998244353998244353998244353 的个数。

  • AAA 恰好包含 111 的 X1X_1X1​ 份， 222 恰好包含 X2X_2X2​ 份， 333 恰好包含 X3X_3X3​ 份。
  • 相邻元素之间的绝对差最多为 111 。也就是说，对于满足 1≤i≤X1+X2+X3−11 \leq i \leq X_1+X_2+X_3-11≤i≤X1​+X2​+X3​−1 的每一个整数 iii ，我们都有 ∣ai+1−ai∣≤1|a_{i+1} - a_i| \leq 1∣ai+1​−ai​∣≤1 。

分析：

相邻元素绝对值差最多为1，也就是说对于1和3来说，两侧要么相同为1或3，要么为2，也就是说2将1和3隔开了，那么这就转化为了一个数学题。

思路：

由于如果将2视作隔板，插入1和3的序列中，我们无法统计1和3相邻的位置有多少（因为这些位置必须放入非0个2），所以我们转换思路，在全是2的序列中插入连续的1或者3。
对于X2X_2X2​个2，我们有X2+1X_2+1X2​+1个位置可以插入1或3，因此需要在其中选择iii个位置，即(X2+1i)\binom{X_2+1}{i}(iX2​+1​)，随后在这iii个位置中，我们需要选择jjj个位置插入1，i−ji-ji−j个位置插入3，也就是(ij)\binom{i}{j}(ji​)，两者是相似的情况，这里只解释选择jjj个位置插入1有多少种情况。
对于X1X_1X1​个1，我们需要将他们分为jjj份，也就是说我们要选择j−1j-1j−1个隔板，而一共有X1−1X_1-1X1​−1个可以做隔板的位置，那么一共有(X1−1j−1)\binom{X_1-1}{j-1}(j−1X1​−1​)。
同理对于3来说共有(X3−1i−j−1)\binom{X_3-1}{i-j-1}(i−j−1X3​−1​)。
所以最终的答案为ans=∑i=1X2+1(X2+1i)∑j=0i(ij)(X1−1j−1)(X3−1i−j−1)ans=\sum\limits_{i=1}^{X_2+1}\binom{X_2+1}{i}\sum\limits_{j=0}^{i}\binom{i}{j}\binom{X_1-1}{j-1}\binom{X_3-1}{i-j-1}ans=i=1∑X2​+1​(iX2​+1​)j=0∑i​(ji​)(j−1X1​−1​)(i−j−1X3​−1​)。

化简结果为∑k=1X2+1(X2+1k)(X1−1k−1)(X2+X3−kX3)\sum\limits_{k=1}^{X_2+1}\binom{X_2+1}{k}\binom{X_1-1}{k-1}\binom{X_2+X_3-k}{X_3}k=1∑X2​+1​(kX2​+1​)(k−1X1​−1​)(X3​X2​+X3​−k​)。
